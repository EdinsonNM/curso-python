# Proyecto de Ejercicios de Python

Este proyecto contiene una serie de ejercicios prácticos sobre el uso de módulos en Python, organizados en la carpeta `modules`. Cada ejercicio está diseñado para enseñar diferentes aspectos de la importación de módulos, el uso de `__main__`, y la estructura de paquetes en Python.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

```
curso-python
├── modules
│   ├── ejercicio_01_importacion_basica
│   ├── ejercicio_02_import_as_y_from
│   ├── ejercicio_03_paquetes
│   ├── ejercicio_04_uso_de_main
│   ├── ejercicio_05_main_como_punto_de_entrada
│   └── ejercicio_06_ejecucion_de_modulo
├── pyproject.toml
└── README.md
```

## Descripción de los Ejercicios

1. **Ejercicio 01: Importación Básica**
   - **Descripción:** Este ejercicio introduce la importación de funciones y clases desde un módulo.
   - **Ubicación:** `modules/ejercicio_01_importacion_basica/README.md`

2. **Ejercicio 02: Importar como y desde**
   - **Descripción:** Se explora el uso de importaciones específicas y alias.
   - **Ubicación:** `modules/ejercicio_02_import_as_y_from/README.md`

3. **Ejercicio 03: Paquetes**
   - **Descripción:** Este ejercicio enseña cómo organizar el código en paquetes.
   - **Ubicación:** `modules/ejercicio_03_paquetes/README.md`

4. **Ejercicio 04: Uso de __main__**
   - **Descripción:** Se muestra cómo utilizar el bloque `if __name__ == "__main__":` para ejecutar scripts.
   - **Ubicación:** `modules/ejercicio_04_uso_de_main/README.md`

5. **Ejercicio 05: Main como Punto de Entrada**
   - **Descripción:** Este ejercicio profundiza en el concepto de `main` como punto de entrada de un programa.
   - **Ubicación:** `modules/ejercicio_05_main_como_punto_de_entrada/README.md`

6. **Ejercicio 06: Ejecución de Módulo**
   - **Descripción:** Se enseña cómo ejecutar un módulo directamente desde la línea de comandos.
   - **Ubicación:** `modules/ejercicio_06_ejecucion_de_modulo/README.md`

## Cómo Ejecutar los Ejercicios

Para ejecutar cada ejercicio, navega a la carpeta correspondiente y ejecuta el archivo `main.py` o el archivo designado como punto de entrada. Asegúrate de tener Python instalado en tu sistema.
